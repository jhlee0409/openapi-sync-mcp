# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-19

### Added

- Initial release
- `oas_parse` - Parse OpenAPI specs with pagination and caching
- `oas_deps` - Query dependency graph for schema impact analysis
- `oas_diff` - Compare spec versions with breaking change detection
- `oas_status` - Get cached spec status
- `oas_generate` - Generate code for TypeScript, Rust, Python

### Supported Platforms

- macOS (x64, arm64)
- Linux (x64, arm64)
- Windows (x64)
